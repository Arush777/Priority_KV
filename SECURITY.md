# Security Policy

PriorityKV is a research prototype, not a hardened production serving system.

Report vulnerabilities privately to the repository owner through GitHub's security
advisory feature. Do not include credentials, private model tokens, or exploit details in
public issues. The maintainers will acknowledge a report when available and coordinate a
fix or disclosure timeline based on severity.

Only the latest `main` branch is maintained. Model files, CUDA toolchains, FlashInfer,
Transformers, vLLM, and other third-party dependencies follow their own security policies.
